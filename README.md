# risk-profile

## Requirements

- Python 3.7 (it can be used with pyenv)
- pipenv

## Installation
```bash
$ pipenv install --python 3.7
```
  
## Running
```bash
$ ./run
```

# Development
If you want to use to install dev dependencies such as jedi and flake8 just run
```bash
$ pipenv install --dev --python 3.7
```
